Создайте приложение, которое использует RecyclerView для отображения списка рецептов.
Каждый элемент списка отображает название рецепта с кратким описанием.
Используйте разный текст в TextView для названия и описания рецепта.

Когда пользователь выбирает рецепт (элемент в списке) - запустите действие, которое отображает полный текст рецепта.

Вы можете использовать текст-рыбу для отображения полного описания рецептов.
При желании добавьте изображение для готового блюда в каждый рецепт.

Нажатие кнопки назад возвращает пользователя к списку рецептов.

Добавьте меню с возможностью поиска рецепта по названию с отображением подходящих рецептов
по мере ввода текста.

При обновлении списка подходящих запросу рецептов рекомендуется использовать DiffUtil.

Дополнительный бонус за использование фрагментов для адаптации отображения к  режиму планшета.
