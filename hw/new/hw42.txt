Задание 1 Функции агрегирования. Объединения.

Разработать базу данных хранящую информацию о пассажирских перевозках

База должна позволять хранить данные:
•	Информация о доступном транспорте
o	Номера 
o	Марки транспорта
o	Количество мест в транспорте

•	Информацию о водителях
o	ФИО
o	Класс
o	Дата рождения
o	Контактные данные
o	Дополнительная информация 

•	Данные о перевозках
o	Тип перевозки (рейсовые, заказные и т д)
o	Километраж
o	Количество пассажиров
o	Количество планируемого (затраченного) топлива

•	Данные о проданных билетах
o	Общее количество билетов на рейс
o	Количество проданных билетов на рейс
o	Зарезервированные билеты
o	Данные о цене билетов

Запросы:
Реализовать следующие запросы:
•	Выбрать весь свободный автотранспорт, находящийся на станции в данный момент
•	Выбрать всех не занятых в рейсах водителей и отсортировать по классности
•	Выбрать все поездки (рейсы) за определенный период времени
•	Выбрать информацию о проданных билетах за определенный период
•	Выбрать все рентабельные рейсы. (Рейсы, которые были выгодны автостанции и принесли доход)

Прислать файл с sql запросами и базу данных в формате sqlite.
