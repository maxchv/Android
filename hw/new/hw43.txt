Задание:

Закончить пример приложение ToDoList с применением
контент провайдера начатого на занятии.

Реализовать:
1. Метод update класса ToDoListProvider
2. Отобразить все записи в ListView через SimpleCursorAdapter
3. Реализовать функцию добваления записи через диалоговое окно
4. Реализовать удаление записи при отметке чекбокса _DONE.

На повышенную оценку:
1. Добавить вторую версию базы данных.
2. Прописать корректынй скрипт обновления таблицы базы данных в метод onUpgrade класса DbHelper
   с учетом поля deleted, в котором будет храниться маркер удаленной записи
3. По факту записи удалятся не будут, а будут помечаться маркером, что они удалены.
4. При отображении записей в списке выводить не удаленные записи.
